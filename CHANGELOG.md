# 更新日志 / Changelog

## [v1.1.0] - 2026-04-03

### 新增功能 / New Features

- **Nothing 风格全新 UI**
  - 点阵网格背景
  - 浅灰色背景设计
  - 极简顶部导航栏
  - 全屏 Hero 区域
  - 悬浮操作按钮

  - 使用 Space Grotesk 作为英文字体

  - 添加 `npm start` 命令启动服务器

- **布局**
  - 顶部固定导航栏（毛玻璃效果）

  - 支持拖拽上传
  - 实时预览

- **预览区域**
  - 4 列网格布局
  - 悬浮删除按钮
  - 编号显示

- **按钮样式**
  - 胶囊形状按钮

  - 禁用状态样式
