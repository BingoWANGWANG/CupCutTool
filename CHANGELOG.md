# 更新日志 / Changelog

### 界面优化

- 批量结果移至右侧面板顶部，固定显示

- 修复"处理中"状态与批量结果重叠问题
- 修复下载全部按钮宽度过大问题

- 添加 nodemon 支持，修改代码自动重启服务器

## [v1.3.0] - 2026-04-04

### 新增功能

- **批量处理功能**
  - 支持上传整个文件夹
  - 图片按名称自动排序
  - 每 4 张图片为一组生成一张拼图
  - 批量结果展示网格，每行 4 张
  - 支持单独下载和批量下载全部

- **裁剪顶部位置设置**
  - 可调整裁剪起始位置（默认 0.4）
  - 支持运行时动态调整
  - 修改后自动重新生成拼图

### 修复问题

- 修复文件夹上传重复弹窗问题
- 修复批量处理后无法调整裁剪参数的问题
- 修复上传文件夹按钮大小异常问题

### 代码优化

- 重构批量处理逻辑
- 优化前端代码结构

## [v1.2.0] - 2026-04-03

### 新增功能

- **两栏布局设计**
  - 左侧：上传区 + 缩略图列表 + 操作按钮
  - 右侧：大块预览区域显示生成结果
  - 整体居中，最大宽度 1200px

- **下载按钮**
  - 生成结果后显示下载按钮
  - 点击即可下载 PNG 图片到本地

- **Loading 状态**
  - 处理中显示加载动画
  - Loading 居中显示

### 页面优化

- 布局更紧凑，减少空白
- 预览列表改为左侧小缩略图 + 文字形式
- 结果区域最小高度 400px
- 响应式设计适配移动端

### 修复问题

- 修复结果区域显示 bug
- 优化生成后图片显示逻辑
