# 更新日志 / Changelog

## [v1.2.0] - 2026-04-03

### 新增功能

- **两栏布局设计**
  - 左侧：上传区 + 缩略图列表 + 操作按钮
  - 右侧：大块预览区域显示生成结果
  - 整体居中，最大宽度 1200px

- **下载按钮**
  - 生成结果后显示下载按钮
  - 点击即可下载 PNG 图片到本地

- **Loading 状态**
  - 处理中显示加载动画
  - Loading 居中显示

### 页面优化

- 布局更紧凑，减少空白
- 预览列表改为左侧小缩略图 + 文字形式
- 结果区域最小高度 400px
- 响应式设计适配移动端

### 修复问题

- 修复结果区域显示 bug
- 优化生成后图片显示逻辑
